# Pitch et arguments de vente

Les phrases qui vendent. À utiliser sur le site portfolio, dans les emails de devis, en visio brief, et partout où on parle aux clients. Cohérence du discours = crédibilité.

> Doublon dans `portfolio/documentation/produit/pitch.md` pour le projet web.

## Pitch principal de l'offre

**En une phrase** :
> *« Je conçois des sites web sur mesure en Next.js — pas de WordPress, pas d'abonnement à vie, code à vous. »*

**En 3 phrases** :
> *« Je suis développeur freelance, 3 ans d'expérience dans des grands groupes (TotalEnergies, BNP, ADP). Je crée des sites web pour artisans, commerçants et entreprises qui veulent une vraie présence en ligne. Pas de template, pas de plugin, pas de commission sur vos ventes — juste du code propre, livré rapidement, qui vous appartient. »*

## Pitch par formule

### Essentiel — Site vitrine 1 page

> *« Une présence pro sans abonnement à vie. Code à vous, hébergement inclus, zéro WordPress. Le coût d'un an de Wix, payé une fois. »*

**Arguments à dérouler en visio** :

- Vous existez sur Google en 5 jours
- Aucun abonnement mensuel à payer (Vercel gratuit, juste le domaine à 10-15€/an)
- Site rapide (Lighthouse 90+ même sur l'offre de base)
- Si un jour vous voulez changer de prestataire, le code est à vous, vous partez avec
- On n'est pas chez Wix : pas de pub, pas de mention « Made with Wix »

### Standard — Site vitrine multi-pages ⭐

> *« Un site qui travaille pour vous. Le SEO local + Google Analytics + Google Maps font venir des prospects qualifiés sur Google sans payer de pub. Pendant que vous bossez, votre site vous trouve des clients. »*

**Arguments à dérouler en visio** :

- Vous apparaissez dans les recherches Google de votre secteur (« plombier Lyon », « avocat divorce Paris »)
- Vous voyez exactement combien de personnes visitent votre site et d'où elles viennent (Google Analytics)
- Votre fiche Google Maps est optimisée pour ressortir dans les recherches locales
- Les textes sont rédigés pour convaincre ET pour être bien compris par Google
- Vous pouvez voir ce que les visiteurs cherchent et adapter votre offre

**À éviter** : *« Le site va vous ramener X clients par mois »*. Ne jamais promettre de chiffres, le SEO dépend de la concurrence et du secteur.

### Premium — Site sur mesure

> *« Un site qui vous distingue. Design 100% custom, niveau agence, sans le prix d'agence. Vous gérez votre contenu en autonomie via le CMS, vos visiteurs voient un site qui n'existe nulle part ailleurs. »*

**Arguments à dérouler en visio** :

- 3 directions visuelles proposées en maquette, vous choisissez celle qui vous parle le mieux
- Animations sur mesure qui renforcent votre identité de marque
- Vous modifiez vos textes, photos, contenus en autonomie via Sanity (pas besoin de me rappeler à chaque modif)
- Performance technique irréprochable : Lighthouse 95+, mobile parfait, SEO avancé
- 1 mois de support inclus pour ajuster ce qui peut l'être après livraison
- Une agence facturerait 5 000-8 000€ pour ce niveau

## Comparatifs face à la concurrence

### Toi vs Shopify (e-commerce)

| | Toi (Essentiel + boutique avec panier = 1 180€) | Shopify Basic |
|---|---|---|
| Prix de départ | 1 180€ une fois | 32€/mois |
| Coût sur 3 ans | 1 180€ | 1 152€ |
| Coût sur 5 ans | 1 180€ | 1 920€ |
| Commission par vente | 0% (Stripe seul : 1.5% + 0.25€) | 2% Shopify + 1.6% Stripe |
| Performance / SEO | Lighthouse 95+ | Limité par le thème |
| Tu pars un jour ? | Le code t'appartient | Tu perds tout |
| Aspect | Custom unique | Reconnaissable comme Shopify |

**Phrase clé** :
> *« Le site se rembourse en 2-3 ans face à Shopify. Au-delà, c'est de l'argent que vous économisez chaque mois. Et vous ne payez pas une commission Shopify sur chacune de vos ventes. »*

### Toi vs Wix / Squarespace (vitrine)

| | Toi (Standard = 990€) | Wix Premium |
|---|---|---|
| Prix de départ | 990€ une fois | 16€/mois |
| Coût sur 5 ans | 990€ | 960€ |
| Personnalisation | Vraie charte graphique | Limité aux templates |
| Performance | Lighthouse 90+ | Souvent 50-70 |
| SEO | Schema.org, Local Business | Basique |
| Pub Wix sur le site | Aucune | Présente sur l'offre gratuite |
| Vous partez ? | Code à vous | Tout reste chez Wix |

**Phrase clé** :
> *« Pour le même budget sur 5 ans, vous avez un site sur mesure plutôt qu'un template Wix reconnaissable. Et si un jour vous voulez changer, vous emmenez votre site avec vous. »*

### Toi vs WordPress (vitrine ou e-commerce)

| | Toi (Standard = 990€) | WordPress avec dev freelance |
|---|---|---|
| Prix initial | 990€ | 1 500-3 000€ (selon dev) |
| Maintenance | 50€/mois (optionnel) | 50-150€/mois (obligatoire) |
| Sécurité | Aucune faille connue | Plugins à patcher constamment |
| Performance | Lighthouse 90+ | Souvent < 60 |
| Mises à jour | Automatiques (Next.js, Vercel) | Manuelles (WordPress, plugins) |
| Hébergement | Inclus (Vercel gratuit) | OVH, o2switch... 5-15€/mois |

**Phrase clé** :
> *« WordPress c'est gratuit en apparence, mais vous payez la maintenance, l'hébergement, les plugins payants et les corrections de failles toute l'année. Mon site est plus rapide, plus sûr, et la maintenance est optionnelle. »*

### Toi vs No-code (Webflow, Framer)

> *« Webflow et Framer sont excellents pour des designs simples. Mais dès qu'on veut une intégration custom (paiement Stripe spécifique, animation 3D, base de données structurée, multilingue propre), on bute sur les limites. C'est là que ma valeur entre en jeu : je fais ce que le no-code ne peut pas faire bien. »*

À utiliser uniquement si le client compare avec ces outils.

## Objections fréquentes et réponses

### « C'est cher »

> *« Mes prix sont volontairement positionnés en dessous du marché parisien (qui démarre à 2 500€ pour un site équivalent). Je travaille seul, sans agence, ce qui me permet d'être compétitif. Comparez avec un abonnement Wix sur 3 ans : on est dans les mêmes prix, sauf que mon site vous appartient. »*

### « Pourquoi pas WordPress, c'est gratuit »

> *« WordPress n'est pas gratuit. Vous payez l'hébergement (5-15€/mois), les plugins payants (Yoast, formulaires, sécurité = 100-300€/an), et la maintenance pour patcher les failles. Sur 3 ans, vous êtes au-dessus de mon prix, avec un site plus lent et moins sécurisé. »*

### « Je peux le faire moi-même avec Wix »

> *« Bien sûr, et si vous avez le temps et l'envie d'apprendre, c'est une option valable. Mais comptez 30-40h pour faire quelque chose de propre, et le résultat sera reconnaissable comme un Wix. Vous pouvez aussi continuer à vous concentrer sur votre métier et avoir un site fait par un pro. »*

### « Ça prend combien de temps ? »

> *« 5 jours pour Essentiel, 10 jours pour Standard, 3 semaines pour Premium. Le compteur démarre une fois que j'ai vos textes et photos. Si vous m'envoyez tout demain, votre site est en ligne la semaine prochaine pour Essentiel. »*

### « Et si je veux modifier après livraison ? »

> *« Trois options :
> 1. Vous prenez l'option CMS (à partir de 150€) et vous modifiez vous-même textes et photos
> 2. Vous prenez la maintenance mensuelle (50€/mois) qui inclut 1h de modifs
> 3. Vous me payez à la demande (60€/h) pour des modifs ponctuelles »*

### « Vous avez des références ? »

> *« J'ai bossé pour TotalEnergies, BNP Paribas et Aéroports de Paris en CDI. Vous pouvez voir mon portfolio sur leohengebaert.fr. Pour les sites freelance, je peux vous montrer 2-3 réalisations récentes. »*

### « C'est quoi la différence avec une agence ? »

> *« Une agence c'est 3-5 personnes (commercial, chef de projet, designer, dev), donc le prix doit couvrir 5 salaires. Moi je suis seul, je m'occupe de tout, vous avez un seul interlocuteur, et je facture moitié moins. La contrepartie : je prends moins de gros projets, donc je suis très sélectif. »*

### « Et si je ne suis pas content du résultat ? »

> *« Le process est conçu pour éviter ça :
> 1. Brief en visio avant de coder une ligne
> 2. Maquette validée avant de poursuivre
> 3. Itérations incluses dans la formule
> 4. Acompte 50%, solde à la livraison
> Si malgré tout vous n'êtes pas satisfait, on en parle. Mais en 3 ans, ça n'est jamais arrivé. »*

## Ton et posture

- **Direct** : pas de jargon technique, des phrases courtes
- **Honnête** : si une option n'est pas pertinente pour le client, le dire (ne pas vendre du SEO avancé à un coiffeur de quartier qui n'en a pas besoin)
- **Confiant** : 3 ans dans des grands groupes, c'est de l'expérience solide
- **Pas vendeur** : le client doit avoir l'impression d'un échange, pas d'un pitch commercial
- **Pédagogue** : expliquer ce qu'est un CMS, pourquoi WordPress c'est risqué, etc. Le client achète mieux quand il comprend

## Ce qu'on ne dit JAMAIS

- *« Je vais vous ramener X clients par mois »* (impossible à promettre)
- *« Je suis le moins cher »* (positionnement perdant)
- *« C'est urgent, décidez-vous vite »* (manipulation)
- *« Tout est négociable »* (on perd la valeur perçue)
- *« Je ferai gratuitement ces petits trucs »* (engrenage)
- *« WordPress c'est nul »* (trop tranchant, dire « moins adapté à votre besoin »)
