> Site d'une agence de voyages

**Description**:
Ce projet est un site web pour une agence de voyages fictive. Il présente diverses fonctionnalités telles qu'un menu de navigation, une section d'accueil avec une vidéo, une liste de destinations de voyage et des boutons interactifs.

**Technologies Utilisées**:

- React
- React Router
- CSS : L'utilisation de variables CSS (:root) permet une gestion efficace des couleurs et d'autres propriétés réutilisables dans l'ensemble du site. Les classes de boutons (btn--primary, btn--outline, etc.) permettent une personnalisation facile des styles des boutons dans tout le site. Les styles des cartes sont conçus pour donner un aspect attrayant aux destinations de voyage, y compris des effets de survol pour l'interactivité. Les requêtes médias sont utilisées pour garantir que le site est **responsive à 100%**, améliorant ainsi l'expérience utilisateur sur les appareils mobiles et de bureau.

**Fonctionnalités Clés**:

1. **Menu de Navigation**: Un menu de navigation réactif qui s'ouvre et se ferme lorsqu'on clique sur l'icône de la barre de menu. Il permet aux utilisateurs de naviguer vers différentes sections du site.

2. **Section d'Accueil (HeroSection)**: Une section d'accueil dynamique avec une vidéo de fond. Elle présente un message accrocheur et des boutons call to action pour encourager les visiteurs à explorer le site.

3. **Liste de Destinations (Cards)**: Une liste de destinations de voyage avec des images, des descriptions et des étiquettes pour indiquer le type d'aventure. Les visiteurs peuvent cliquer sur ces destinations pour en savoir plus.

4. **Bouton "S'inscrire"**: Un bouton "S'inscrire" qui redirige les utilisateurs vers la page d'inscription.

**Apprentissage et Réalisations**:

- Apprentissage de la création d'une interface utilisateur réactive en utilisant React.
- Utilisation de React Router pour gérer la navigation entre différentes pages.
- Mise en place d'un menu de navigation réactif avec des icônes de menu hamburger.
- Intégration d'une vidéo de fond dans la section d'accueil.
- Création de composants réutilisables, tels que des boutons (Button) et des cartes (CardItem).

**Prochaines Étapes**:

- Ajouter des pages supplémentaires pour chaque destination de voyage avec des détails complets.
- Intégrer un backend pour gérer les réservations de voyage et les commentaires des utilisateurs.
- Mettre en œuvre un système d'authentification pour permettre aux utilisateurs de se connecter et de gérer leurs réservations.
- Améliorer le design et l'esthétique du site web pour le rendre plus attractif.
